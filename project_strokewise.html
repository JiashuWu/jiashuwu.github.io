About Strokewise

Strokewise is a website which can help people to gain a deeper understanding about the stroke disease. By presenting some introduction of the disease, its symptons, its cause, and a introduction to the related treatment, we help people to be aware of this disease and stick to a healthy lifestyle.

Changelog


A note-taking tool combined with remain days functionality
Record event name, event starting date, finishing date, creating date, event content and event classification, and calculate the remain days automatically.
Display the days remain for each event
Support three classifications for events (Home, Work and Others)
Support two display modes (Event-list Mode, Calendar Mode)
Customise the app with 30 pre-defined theme colours.
Support five languages (English (GB), Simplified Chinese, Traditional Chinese, Japanese, French)
Allow sorting the events in specified order (Order by creating time in ascending or descending order, order by remain days in descending order)
Allow searching by event name and event content.
Allow displaying events with certain constraints (Due in next fourteen days, already passed the due date)
Support notifications of the number of event due on each day at a fixed time.
Automatically save the settings for user (Sorting order setting, notification setting, colour setting)
Automatically save the language setting at the last shut down of the app
